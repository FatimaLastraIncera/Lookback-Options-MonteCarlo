unsigned long seed )

Compute option price and Greeks using Monte Carlo.

Delta and Vega are computed pathwise; Gamma, Rho and Theta use central finite differences with common random numbers for variance reduction.

Parameters
baseBase Monte Carlo parameters.
bumpSBump for underlying price (Gamma calculation).
bumpRBump for interest rate (Rho calculation).
bumpTBump for maturity (Theta calculation).
seedRNG seed for reproducibility.
Returns
Greeks structure with computed sensitivities.
